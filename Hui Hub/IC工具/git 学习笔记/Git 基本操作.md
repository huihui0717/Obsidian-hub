## 一、Git基础
### 1.1 获取 Git 仓库

#### 1.1.1 将现有项目导入到 Git 中
- 在现有项目的目录中初始化仓库，[[Git 命令详解#git init]]
```
$ git init
```
- 使用 [[Git 命令详解#git add|git add]]，[[Git 命令详解#git commit|git commit]] 命令，跟踪项目文件并提交，
```
$ git add ...     // 第一次跟踪
$ git add ...     // 第二次暂存
$ git commit -m "简短说明"
```
